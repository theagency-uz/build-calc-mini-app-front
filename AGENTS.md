## Глобальная структура приложения

В приложении должны быть 3 глобальные вкладки:

-  Грунт;
-  Пол;
-  Клей.

Каждая вкладка содержит форму расчета. После нажатия на кнопку расчета, если все обязательные поля заполнены и валидны, должен открываться экран результата расчета.

## Вкладка «Грунт»

Форма должна содержать поля:

-  Продукт — Select input;
-  Разбавление (концентрат) — Select input;
-  Площадь (m2) — Number input;
-  Запас материалов (%) — single range slider от 0% до 100%;
-  Кнопка «Рассчитать расход».

## Вкладка «Пол»

Форма должна содержать поля:

-  Продукт — Select input;
-  Толщина слоя (мм) — Number input;
-  Площадь (m2) — Number input;
-  Запас материалов (%) — single range slider от 0% до 100%;
-  Кнопка «Рассчитать расход».

## Вкладка «Клей»

Форма должна содержать поля:

-  Вид покрытия — Select input;
-  Марка клея — Select input;
-  Тип основания пола — Select input;
-  Под типом основания пола должна отображаться подсказка, привязанная к выбранному значению в Select input. Пример: «Грунтовка: 044 Europrimer Multi или 045 Eurobond»;
-  Шпатель — Select input;
-  Площадь (m2) — Number input;
-  Кнопка «Рассчитать расход».
